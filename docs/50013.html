Removal Code: 50013

What is RC:50013?

Removal Code 50013 occurs when SPRDZY has insufficient permissions to function properly and affects other servers. SPRDZY requests Administrator. If you do not accept that, that's fine! However, there's a baseline of permissions SPRDZY does need to prevent crashes and - as a result - RC:50013.

By default, SPRDZY needs the following permissions:

• Send Messages (duh!)

• Create Invites

• Ban Members

• Kick Members

• Timeout Members

• Create Expressions

• Manage Members

Yes, for optional features, more permissions are required. This is why we simply just recommend Administrator.


What next?

If you are a partner, you have 7 days to resolve the problem and invite SPRDZY again before we revoke the partnership. For regular users, just fix the permissions. If we have to assign RC:50013 multiple times, we may blacklist you or your server.
